Class ComponentFactory

Factory for microservice components. It registers component classes, locates classes by descriptors and creates component instances. It also supports inheritance from other factories.

In the future we can add an ability to dynamicaly load component libraries from component config. That will allow throw place into microservice container pretty much anything at deployment time.
