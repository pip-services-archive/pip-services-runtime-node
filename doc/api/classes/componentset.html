Class ComponentSet

A list with references to all microservice components. It is capable of searching and retrieving components by specified criteria.

getOnePrior

  • getOnePrior(component: IComponent, descriptor: ComponentDescriptor): IComponent
    • Defined in ComponentSet.ts:232

    Gets a component instance that matches the specified descriptor defined before specified instance. If nothing is found it throws a configuration error. This method is used primarily to find dependencies between business logic components in their logical chain. The sequence goes in order as components were configured. The descriptor is used to specify number of search criteria or their combinations:

    • By category

    • By logical group

    • By functional type

    • By implementation version
    For instance, quite often the descriptor will look as "logic / group / / "

    throws

    MicroserviceError when no components found

    Parameters

    • component: IComponent

      a component that searches for prior dependencies

    • descriptor: ComponentDescriptor

      a component descriptor as a search criteria

    Returns IComponent

    a found component instance
